# Tech Stack

## Language(s)
- HTML5, CSS3, Vanilla JavaScript (ES6+)

## Framework(s)
- None — plain web stack, no build step required

## Runtime
- Any modern browser (Chrome, Safari, Firefox, Edge)
- Installable as a PWA via `manifest.json`

## Package manager
- None

## Database / storage
- `localStorage` (client-side, no backend)

## Key libraries
- None — zero dependencies

## Common Commands

```bash
# Open directly in browser (no server needed)
# Just open index.html in a browser

# Or serve locally with Python
python -m http.server 8080

# Or with Node (if installed)
npx serve .
```
